Exemplos de perguntas uteis para a IA:

- O que esse erro quer dizer?
- Explica esse codigo linha por linha
- Como faco essa tarefa em Python?

- Meu codigo esta assim, o que ta errado?

## Exercicios da aula

### 1. ola.py

Primeiro contato com Python: imprime mensagens e pede o nome do usuario.

### 2. variaveis.py

Mostra variaveis com tipos basicos:

- str
- int
- float
- bool

### 3. listas_dicts.py

Trabalha com:

- lista com ferramentas de cyber
- dicionario com dados pessoais

### 4. condicionais.py

Pede uma nota de 0 a 10 e informa:

- aprovado
- recuperacao
- reprovado

### 5. loops.py

Exemplos praticos com:

- numeros de 1 a 10
- lista de nomes
- tabuada

### 6. funcoes.py

Implementa tres funcoes:

- calcular_area(largura, altura)
- e_maior(idade)
- apresentar(nome, curso)

## Desafios finais

### 1. calculadora.py

Calculadora simples com operacoes basicas.

### 2. lista_compras.py

Permite adicionar itens ate o usuario decidir encerrar.

### 3. quiz.py

Quiz com 5 perguntas e contagem de acertos.

### 4. gerador_senhas.py

Gera senha aleatoria com o tamanho informado pelo usuario.

### 5. organizador.py

Recebe nomes digitados pelo usuario e mostra a lista em ordem alfabetica.
